## ✨ About This Project

Welcome to my personal portfolio website! This is a modern, responsive, and feature-rich portfolio showcasing my work as a **Full Stack Developer & Designer**. Built with React, Vite, and custom CSS, it delivers a smooth, engaging user experience with a focus on performance and aesthetics.

---

## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🎨 **Modern Design** | Clean, professional layout with smooth animations |
| 🌓 **Dark/Light Theme** | User preference-based theme toggle with persistence |
| 📱 **Fully Responsive** | Perfect on desktop, tablet, and mobile devices |
| 🖱️ **Custom Cursor** | Interactive gradient cursor with hover effects |
| ✉️ **Contact Form** | Functional contact form with Web3Forms integration |
| 📄 **Resume Download** | Direct Google Drive resume download link |
| 🔗 **Social Integration** | Active links to GitHub, LinkedIn, X (Twitter) |
| 🎬 **Smooth Animations** | Framer Motion powered scroll and hover effects |
| ⚡ **Fast Performance** | Optimized with Vite for lightning-fast loads |
| 🔍 **SEO Friendly** | Meta tags and semantic HTML structure |
